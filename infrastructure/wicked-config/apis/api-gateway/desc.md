# API Gateway

This is the main API Gateway service for Travel Kingdom. It handles:

- `/search/flights` - Flight search via Duffel/Amadeus
- `/search/hotels` - Hotel search via LiteAPI/Innstant
- `/airports` - Static airport data
- `/airlines` - Static airline data
- `/cities` - Static city data
- `/currencies` - Static currency data
- `/loyalty-programs` - Static loyalty program data
- `/static/suggestions` - Unified search suggestions
- `/auth/*` - Authentication endpoints
- `/bookings/*` - Booking management
- `/wallets/*` - Wallet and payment management
